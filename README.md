# Q&AI Learning

This project is an attempt to make learning quicker, enjoyable and longer-lasting for people who prefer talking over the topics and really understand them rather than learning answers by heart.

## Idea

1. You upload a file - a list of questions, presentations etc. and an LLM analyses the input.
2. It generates a directed graph structure tailored to make the learning process as natural as possible, starting with the core ideas and diving deeper with every step. As a user you are able to add your specific preferences.
3. The graph is displayed and you can start learning! 

## Proposed learning process

1. You choose a question and a chat opens - You can pick a LLM of your choice (**local or API**) and start conversating about the topic.
2. The chatbot evaluates your answers, answers questions, and guides you in the right direction.
3. The conversation is saved for you to be able to come back to it later. 

## Configurability

The LLM is fully configurable, You can give it personalities, traits, and direct commands on what style of answers You would like to receive
